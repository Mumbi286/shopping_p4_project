# Installation requirement for this project
1. In the project directory in the terminal run:(this creates the dependancies): 
    * python -m venv fastapienv
2. To activate the virtual environment: 
    * source fastapienv/bin/activate
3. Install the pip dependencies: 
    * pip install fastapi
4. To install web server for fast api: 
    pip install "uvicorn[standard]"
5. Installing dependecies of password hashing
    * pip install passlib
6. Installing bcrypt
    * pip install bcrypt==4.0.1
7. Returns the secret key 
    openssl rand -hex 32
8. pip install pymysql
9. pip install alembic
10. alembic init alembic

# Running the application
3. To run the application 
    * uvicorn main:app --reload
